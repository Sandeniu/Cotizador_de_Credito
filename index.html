Simulador de crédito - JavaScript

Esta página web usa un algoritmo en JavaScript el cual realiza peticiones http utilizando fetch, para ello se han utilizado los siguientes recursos de JS:

  • Variables del proceso
  • Captura de eventos con su función de respuesta
  • Modificación de DOM, para mostrar resultado de procesamiento
  • Metodos de Búsqueda y filtrado sobre array
  • JSON y Storage. Almacenamiento de datos (clave-valor) y recuperación de ellos
  • Librerías (en este caso toastify y Sweet Alert)
  • Promise, para operaciones asíncronas
  • Ajax y fetch()
  • Consulta de API. En este caso la API consultada es la Tasa Máxima Convencional (TMC)" desde la página cmfchile.cl (Comisión para el mercado financiero (CMF)), la cual es utilizada en el cálculo del indicador CAE, utilizado en créditos

Haz clic en el botón para Simular un crédito
